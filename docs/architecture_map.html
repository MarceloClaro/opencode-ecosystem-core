Instruções de leitura do mapa
1. Escolha o registro

"Leigo" traduz cada camada em linguagem cotidiana. "PhD" mostra os nomes técnicos reais, arquivos e specs correspondentes.

2. Passe o mouse / toque

Cada bloco da mesa é uma camada arquitetural real. Ele se levanta da mesa e o painel à direita lista todos os componentes internos.

3. Mova o cursor pela cena

A mesa inteira inclina levemente seguindo o cursor — é uma perspectiva 3D real (CSS perspective + translateZ), não uma imagem estática.

4. Linhas tracejadas

Representam o fluxo real de delegação: o orquestrador central distribui trabalho para cada camada e recebe de volta reflexões que alimentam a memória metacognitiva.
