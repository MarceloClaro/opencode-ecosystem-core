## 4. Estrutura LaTeX

### 4.1 Comandos Utilizados
- `\chapter{}` para capítulos
- `\section{}` para seções
- `\subsection{}` para subseções
- `\subsubsection{}` para subsubseções
- `\label{}` para referências cruzadas
- `\begin{table}` para tabelas
- `\begin{itemize}` e `\begin{enumerate}` para listas

### 4.2 Exemplo de Estrutura
```latex
\chapter{Introdução}
\label{chap:introducao}

\section{Objetivo e finalidade do projeto}
\label{sec:objetivo-finalidade}

\subsection{Problemas sociais do tema no contexto brasileiro}
\label{subsec:problemas-sociais}

\subsubsection{Descrição do problema}
\label{subsubsec:descricao-problema}
```
