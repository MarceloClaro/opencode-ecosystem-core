\begin{description}
    \item[ABNT NBR ISO/IEC 3173] Norma brasileira que estabelece o vocabulário.
    \item[Digital Twin (Gêmeo Digital)] Representação virtual dinâmica.
    \item[LLM (Large Language Model)] Modelo de Linguagem de Grande Escala.
    \item[OpenCode Ecosystem] Plataforma open-source de orquestração multiagente.
    \item[TDD (Test-Driven Development)] Metodologia de desenvolvimento orientado a testes.
\end{description}
