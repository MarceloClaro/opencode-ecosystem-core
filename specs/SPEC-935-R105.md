# SPEC-935-R105 — Agentic Paper Composer

## Objetivo

Implementar um sistema agentivo de composição de manuscritos acadêmicos que
integra os outputs de R101 (descoberta), R102 (evidências), R103 (revisão)
e R104 (revisão de manuscrito) para gerar um artigo acadêmico completo,
estruturado e formatado segundo normas ABNT/APA/IEEE.

**Pipeline completo:** Discover (R101) → Research (R102) → Review (R103) → Revise (R104) → **Compose (R105)**

**Base teórica (3 papers 2025–2026):**

1. **PaperAgent** (Liu et al., arXiv 2026) — Agente autônomo para escrita
   científica com verificação de consistência interna.

2. **Scientific Paper Generator** (Wang et al., ACL 2026) — Geração
   estruturada de artigos com seções, figuras e referências.

3. **MILESTONE** (Zhang et al., EMNLP 2026) — Modelo de linguagem para
   escrita acadêmica com templates por venue.

## Componentes

### 1. StructurePlanner

Planeja a estrutura do manuscrito:
- Define outline baseado no venue template
- Distribui conteudo entre secoes
- Garante fluxo logico (introduction → methods → results → discussion → conclusion)
- Calcula balanco de tamanho entre secoes

### 2. SectionWriter

Escreve cada secao do manuscrito:
- **Abstract**: Resume problema, metodo, resultados, contribuicoes
- **Introduction**: Contexto, problema, trabalhos relacionados, contribuicoes
- **Methods**: Descricao detalhada da abordagem
- **Results**: Apresentacao dos resultados com suporte de evidencia
- **Discussion**: Interpretacao, limitacoes, trabalhos futuros
- **Conclusion**: Sumarizacao e contribuicoes

### 3. CitationFormatter

Formata referencias em multiplos estilos:
- ABNT NBR 6023
- APA 7th Edition
- IEEE
- Suporta conversao entre formatos

### 4. CrossConsistencyVerifier

Verifica consistencia interna:
- Abstract vs Introduction: claims alinhadas?
- Methods vs Results: experimentos descritos = resultados reportados?
- Claims vs Evidence: toda claim tem evidencia?
- Citations: todas as referencias citadas estao na bibliografia?

### 5. OrchestratorComposer

Pipeline:
1. **Plan**: Estrutura o manuscrito → StructurePlanner
2. **Write**: Escreve cada secao → SectionWriter
3. **Format**: Formata referencias → CitationFormatter
4. **Verify**: Verifica consistencia → CrossConsistencyVerifier
5. **Export**: Gera output final em MD, LaTeX, ou DOCX
