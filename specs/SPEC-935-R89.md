# SPEC-935-R89: Thesis Enrichment via Antigravity Web Search

## Objetivo
Enriquecer teses com pesquisa web real via Antigravity Bridge, encontrando papers acadêmicos, contextos e referências para cada conceito das teses.

## Arquitetura
1. **Antigravity Web Search** — busca real de papers para conceitos-chave
2. **Antigravity Read URL** — leitura de abstracts de papers encontrados
3. **Cache persistente** — evita re-busca dos mesmos conceitos
4. **Fallback** — dados simulados quando offline

## Critérios de Aceitação
1. `synthetic_university/thesis_enricher.py` — módulo de enriquecimento
2. Busca web real por conceitos-chave das teses
3. Extração de abstracts e metadados
4. Cache de resultados por conceito
5. Fallback gracioso quando offline
6. 8+ testes TDD
