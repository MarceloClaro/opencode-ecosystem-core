## 1. Visão Geral

Este ciclo especifica a **avaliação, catalogação e integração** de **56 skills de infraestrutura cloud** e **269 scripts operacionais** provenientes do backup do Antigravity (`antigravity-backup-2026-07-11`) no ecossistema OpenCode.

O backup contém skills especializadas do ecossistema Google Cloud Platform (AlloyDB, Cloud SQL, BigQuery, Dataflow, etc.) que **não existem** no catálogo atual de 160 agentes. A integração destas skills preenche uma lacuna crítica de **operações de infraestrutura de dados em nuvem**.

## 9. Decisões de Design

| Decisão | Justificativa |
|:--------|:-------------|
| Catálogo markdown (não JSON) | Padrão do ecossistema; 160 agentes já usam `.md` |
| Agentes por categoria (não 1:1) | 56 skills → 8 agentes é mais gerenciável que 56 micro-agentes |
| Scripts copiados (não symlinked) | Portabilidade; backup pode ser movido |
| Apache 2.0 mantido | Conformidade legal (licença original do Google) |
| TDD de documentação | Verificar conteúdo sem executar scripts cloud (evita dependências GCP) |
