# SPEC-935-R88: LLM-Powered Evaluation via Antigravity

## Objetivo
Substituir o feedback template por avaliações geradas por LLM real via Antigravity Bridge (delegate task), com fallback para Ollama CLI ou templates.

## Arquitetura
1. **Antigravity delegate** (`task_type="analysis"`) — primeira tentativa
2. **Ollama CLI** (`ollama run tinyllama`) — segunda tentativa (se tinyllama disponível)
3. **Template fallback** — último recurso

## Critérios de Aceitação
1. `synthetic_university/llm_evaluator.py` — módulo com 3 níveis de fallback
2. Quando Antigravity disponível, feedback é contextualizado e rico
3. Métricas de tempo e sucesso registradas
4. Cache de resultados para evitar re-geração
5. 8+ testes TDD com mocks

## Fluxo
Professor + Tese → prompt contextualizado → Antigravity delegate → feedback rico → fallback se falhar
