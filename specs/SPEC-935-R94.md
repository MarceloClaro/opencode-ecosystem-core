# SPEC-935-R94: Synthetic University MCP Server

## Objetivo
Expor todos os módulos da Universidade Sintética como ferramentas MCP (Model Context Protocol), permitindo que qualquer cliente MCP (OpenCode CLI, Antigravity, outros agentes) invoque geração de teses, avaliação, revisão, enriquecimento e publicação.

## Arquitetura
1. **SimpleMCPServer** — mesmo padrão do `mci/mcp_server.py`
2. **8+ ferramentas**: geração, avaliação, enriquecimento, visual, revisão, pacote, novidade, dashboard
3. **Stdio transport** — compatível com OpenCode CLI e qualquer cliente MCP

## Critérios de Aceitação
1. `synthetic_university/mcp_server.py` — servidor MCP dedicado
2. 8 ferramentas registradas com schemas JSON completos
3. Rodável via `python3 -m synthetic_university.mcp_server`
4. Testável via stdio (echo JSON | python3)
5. 8+ testes TDD com simulação de stdio
