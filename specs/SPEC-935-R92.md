# SPEC-935-R92: Submission Package Generator

## Objetivo
Gerar pacote completo de submissão para periódicos Qualis A1: documentos, metadados, compliance check, e estrutura de diretórios.

## Arquitetura
1. **Compliance Check**: verifica requisitos de 3 periódicos Qualis A1
2. **Metadata Generator**: gera metadados (DOI, ORCID, keywords)
3. **Package Builder**: monta diretório com manuscript, figures, cover letter, response to reviewers
4. **Checklist**: verifica completude do pacote

## Critérios de Aceitação
1. `synthetic_university/submission_package.py` — módulo completo
2. Compliance check para 3 periódicos A1
3. Geração de cover letter personalizada
4. Estrutura de diretórios padronizada
5. Checklist de submissão
6. 10+ testes TDD
