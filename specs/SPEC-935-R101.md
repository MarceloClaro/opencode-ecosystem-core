# SPEC-935-R101 — Agentic Science V2 (EvoSci)

## Objetivo

Implementar um sistema multiagente bio-inspirado para descoberta científica autônoma,
integrando papéis especializados (mentor, pesquisador, revisor), memória evolutiva persistente,
operações darwinianas (seleção, crossover, mutação) e engenharia de ambiente (permissões,
artefatos, orçamento, supervisão humana).

**Base teórica (3 papers 2026):**

1. **EvoSci** (Xiong et al., ACL 2026) — Framework multiagente bio-inspirado com mentor,
   researcher, reviewer e grafo de conhecimento para evolução de ideias científicas.
   ([arXiv:2605.24018](https://arxiv.org/abs/2605.24018))

2. **EvoScientist** (arXiv 2026) — Memórias persistentes de ideação e experimentação que
   guiam melhoria contínua em descoberta científica ponta-a-ponta.
   ([arXiv:2603.08127](https://arxiv.org/pdf/2603.08127v1))

3. **EurekAgent** (Xin et al., arXiv 2026) — Engenharia de ambiente em 4 dimensões:
   permissões, artefatos (Git-based), orçamento, humano-no-loop.
   ([arXiv:2606.13662](https://arxiv.org/abs/2606.13662))

## Componentes

### 1. ProblemSpaceConstructor
- MentorAgent analisa base de conhecimento (grafo R99) e constrói clusters de problemas interdisciplinares
- Cada cluster tem embedding semântico + metadados (domínios, viabilidade, novidade potencial)

### 2. RoleBasedAgents

| Agente | Função | Base paper |
|--------|--------|------------|
| `MentorAgent` | Define direção, seleciona clusters, avalia fitness geral | EvoSci |
| `PrimeResearcherAgent` | Lidera pesquisa colaborativa, decompõe tarefas | EvoSci |
| `AssistantResearcherAgent` | Executa sub-tarefas (literatura, experimentos) | EvoSci |
| `ReviewerAgent` | Avalia ideias geradas (novidade, viabilidade, impacto) | EvoSci |
| `EvolutionManagerAgent` | Destila memórias persistentes de ideação + experimentação | EvoScientist |

### 3. EvolutionaryEngine
Operações darwinianas sobre clusters de entidades no grafo de conhecimento:
- **Selection**: Filtra clusters com base em feedback do reviewer
- **Crossover**: Recombina entidades de clusters de alta fitness
- **Mutation**: Injeta entidades novas ou de baixa frequência para evitar convergência prematura
- **Inheritance**: Propaga entidades de alta fitness para próxima iteração

### 4. Persistent Memory Ecosystem
- **IdeationMemory**: Sumariza direções de pesquisa viáveis + registra falhas (EvoScientist)
- **ExperimentationMemory**: Captura estratégias efetivas de processamento e modelagem
- Integração com R97 (HeartbeatReflection + StagnationDetector)

### 5. Environment Engineering
Adaptação do EurekAgent para o ecossistema OpenCode:
- **PermissionsEngine**: Isolamento Docker, evaluator oculto, acesso GPU default-deny
- **ArtifactEngine**: Workspaces Git-based, ranked solution history, caches de busca
- **BudgetEngine**: Limites de tempo e custo de API por sessão
- **HITLEngine**: Terminal TUI + web monitor para supervisão e intervenção
