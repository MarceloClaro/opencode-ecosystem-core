# SPEC-935-R86: Academic Integration (arXiv + Semantic Scholar + Sci-Hub CLI)

## Objetivo
Integrar fontes acadêmicas externas para enriquecer as teses com referências reais, dados de citação e verificação de acesso.

## Integrações

### 1. arXiv API
- **Endpoint**: `http://export.arxiv.org/api/query`
- **Função**: Buscar papers reais que correspondam aos conceitos da tese
- **Formato**: XML over HTTP, paginação padrão
- **Cache**: Resultados armazenados localmente para evitar re-busca

### 2. Semantic Scholar API
- **Endpoint**: `https://api.semanticscholar.org/graph/v1/paper/search`
- **Função**: Obter contagem de citações, influência, venues
- **Autenticação**: API key opcional (via env `S2_API_KEY`)
- **Rate limit**: 1 req/s sem chave, 100 req/s com chave

### 3. Sci-Hub (via sci-hub CLI / HTTP)
- **Função**: Verificar disponibilidade de papers em acesso aberto
- **Domínios**: sci-hub.se, sci-hub.ru, sci-hub.st (fallback automático)
- **Estratégia**: Tentativa de fetch do PDF via DOI — se retorna 200, paper está disponível
- **CLI**: Função que simula sci-hub CLI via HTTP requests diretos

## Critérios de Aceitação
1. `synthetic_university/academic_integration.py` — módulo com as 3 integrações
2. arXiv search: retorna papers reais para consultas de conceitos
3. Semantic Scholar: retorna citation count + venue + influência
4. Sci-Hub: verifica disponibilidade via DOI check
5. `LiteratureBacking` score anexado a cada tese validada
6. 10+ testes TDD (com mock para APIs externas)
