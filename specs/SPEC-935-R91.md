# SPEC-935-R91: Multi-LLM Blind Peer Review

## Objetivo
Simular revisão cega por pares (double-blind) usando múltiplos LLMs com perfis de revisores diversos, gerando scores, comentários estruturados e decisão editorial.

## Arquitetura
1. **3 revisores** com perfis acadêmicos distintos
2. **Blind review**: tese anonimizada (sem autor)
3. **Estrutura**: score 1-10, summary, strengths, weaknesses, suggestions
4. **Agregação**: média, consenso, recomendação editorial
5. **Cache**: revisões são cacheadas por hash da tese

## Critérios de Aceitação
1. `synthetic_university/peer_review.py` — sistema de revisão
2. Revisão com 3 perfis de professores distintos
3. Estrutura de review: score + summary + strengths + weaknesses + suggestions
4. Relatório agregado com decisão editorial (Accept / Minor Revision / Major Revision / Reject)
5. Cache por hash de conteúdo
6. Geração de PDF/texto do review completo
7. 12+ testes TDD
