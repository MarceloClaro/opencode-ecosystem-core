# SPEC-935-R93: Academic Novelty Analysis

## Objetivo
Analisar a novidade acadêmica de cada tese comparando com a literatura real, gerando score de originalidade, mapa de contribuições e relatório de posicionamento.

## Arquitetura
1. **Web Search**: busca por literatura similar via OpenCode CLI
2. **Novelty Scoring**: originalidade, não-trivialidade, impacto potencial
3. **Contribution Map**: o que a tese adiciona vs o que já existe
4. **Positioning Report**: onde a tese se situa no estado da arte

## Critérios de Aceitação
1. `synthetic_university/novelty_analysis.py` — módulo completo
2. Busca por literatura similar via web
3. Score de novidade (0-100)
4. Mapa de contribuições
5. Relatório de posicionamento
6. 8+ testes TDD
