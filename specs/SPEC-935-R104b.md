# SPEC-935-R104b — Pip-Installable Packages from R101–R103

## Objetivo

Extrair os módulos R101 (Agentic Science V2), R102 (Deep Research Agent) e
R103 (Agentic Peer Review) como pacotes pip instaláveis independentes, permitindo
que qualquer projeto Python (incluindo o `OpenCode_Ecosystem-main`) os utilize
via `pip install`.

## Pacotes a criar

### 1. `opencode-evosci`

**Módulo:** `agentic_science_v2/`

**Componentes empacotados:**
- `agents.py` — MentorAgent, PrimeResearcherAgent, ReviewerAgent, EvolutionManagerAgent
- `evolutionary_engine.py` — Selection, Crossover, Mutation, Inheritance
- `environment.py` — Permissions, Artifacts, Budget, HITL
- `orchestrator.py` — OrchestratorAgent

**Dependências:** Python ≥ 3.10, sem dependências externas obrigatórias (opcional: numpy)

### 2. `opencode-deep-research`

**Módulo:** `agentic_science_v2/{deep_research.py, evidence_graph.py}`

**Componentes empacotados:**
- `evidence_graph.py` — Entity, Relation, Evidence, EvidenceGraph, path-finding
- `deep_research.py` — KnowledgeBaseRegistry, BFRSAgent, DFRSAgent, ExecutionSandbox, Orchestrator

**Dependências:** Python ≥ 3.10, opcional: networkx

### 3. `opencode-peer-review`

**Módulo:** `agentic_science_v2/review_agent.py`

**Componentes empacotados:**
- `review_agent.py` — RubricEngine, ReviewLedger, AuditGraph, MultiCriticReviewer, OrchestratorReviewer

**Dependências:** Python ≥ 3.10, `opencode-deep-research` (opcional, para AuditGraph)

## Estratégia de empacotamento

**Opção A (cópia):** Copiar os arquivos do core para cada pacote.
**Opção B (symlink/referência):** Usar `pyproject.toml` com caminho relativo.

**Decisão:** Usar **Opção A** (cópia) para independência total, com script de sync.
