# SPEC-935-R87: Discovery Dashboard (HTML Interativo)

## Objetivo
Gerar um dashboard HTML auto-contido que visualiza todas as métricas do pipeline de descoberta interdisciplinar.

## Seções do Dashboard
1. **Evolução do Pipeline** — Timeline interativa dos rounds R0-R84
2. **Matriz de Proximidade** — Heatmap 11x11 das faculdades
3. **Coerência** — Comparação pré/pós refinamento
4. **Benchmark** — Refined vs Random vs Jaccard
5. **Teses** — Ranking com scores empíricos
6. **Pool de Professores** — Distribuição por faculdade
7. **i18n** — Seletor de idioma (EN/PT-BR)

## Critérios de Aceitação
1. HTML auto-contido (sem dependências externas)
2. Dados embutidos no próprio HTML (sem chamadas de rede)
3. Visualizações com Chart.js (CDN opcional) ou CSS/SVG puro
4. Gerado por script Python a partir dos dados reais
5. 3+ testes TDD

## Arquivos
- `synthetic_university/dashboard_generator.py` — script gerador
- `academic/dashboard/index.html` — dashboard gerado
- `tests/test_dashboard_generator.py` — testes
