# SPEC-935-R90: Visual Abstract Generation via Antigravity Image

## Objetivo
Gerar abstracts visuais (figuras acadêmicas, diagramas) para as teses de maior score usando o Antigravity Image Generator.

## Arquitetura
1. **Antigravity Image** — geração de figuras acadêmicas a partir de descrições das teses
2. **Pipeline de seleção** — top 5 teses ranqueadas por composite_score
3. **Cache** — imagens salvas em academic/visual_abstracts/
4. **Fallback** — diagramas SVG/texto quando image gen falha

## Critérios de Aceitação
1. `synthetic_university/visual_abstract.py` — módulo de geração
2. Geração de figura para cada uma das top 5 teses
3. Cache com hash do conteúdo para evitar re-geração
4. Fallback para diagrama textual quando API falha
5. Integração com o pipeline: visual_abstracts/ indexadas por thesis_id
6. 8+ testes TDD
